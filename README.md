# OpenSpace.Content
Raw game data: YAML configs, glTF models, mod templates

This repository contains uncompressed game data for Open Space:
- `.yaml`: Config and scenario definitions
- `.gltf`: 3D models
- `Mods/`: Mod templates and base overrides

## Structure
- `Base/`: Core game content
- `Mods/`: Mod packs
- `DLC/`: DLC definitions

## Usage
This folder is read directly by both the Unity game and the launcher at runtime. Do not compress or import into Unity.
